# RealityExGuard v1.0.2

RealityExGuard is an educational, open-source C++ Anti-Cheat system.

## Project Structure
- `src/`: Source code for the anti-cheat client.
- `include/`: Header files for module verification and security logic.
- `docs/`: Documentation and update information.
- `Server/`: Server-side simulation logic.
- `VS Files/`: Visual Studio project files.

## Features
- Process blacklist scanning.
- Anti-attachment (debugger detection).
- Game module integrity verification.
- Simulated server-side kick mechanism.
